BASIC INFORMATION:

    1. PLATFORM: Linux
    2. OS VERSION: Mint 17.1 (x64)
    3. BUILD ENVIRONMENT: Make
    4. BUILD ENVIRONMENT VERSION: Make 3.81
    5. LANGUAGE: C++
    6. EXTERNAL LIBRARIES: GL, GLUT, Make

HOW TO BUILD:

    Install GLUT and OpenGL by typing "sudo apt-get install freeglut3-dev" 
    Install Make by typing "sudo apt-get install make"
    
    After installing, compile by typing "make" in the "cs418_mp3_maila2" directory.
    
    After compiling, run with "./mp3"

Controls:

    '1': no textures
    '2': texture, repeatedly pressing '2' iterates through available textures
    '3': environment mapping
    'a': rotate left
    'd': rotate right
    'w': rotate up
    's': rotate down
    'q': translate out
    'e': translate in
