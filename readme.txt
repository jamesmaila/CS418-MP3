Assignment: MP3 - Teapot Contest

To run:
    Use Makefile to compile by typing "make" in a linux terminal.
    The host machine must have OpenGL and GLUT installed.
    After compiling, run with "./mp3"

Controls:

    '1': no textures
    '2': texture, repeatedly pressing '2' iterates through available textures
    '3': environment mapping
    'a': rotate left
    'd': rotate right
    'w': rotate up
    's': rotate down
    'q': translate out
    'e': translate in
